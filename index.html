If Statements

If statements are logic statements used in code to perform tasks. They will be our first tool to use, so we can perform basic programs.
We will learn about the two if statements for now, the if and else statements.
//To use these statements we need to think of if statements as code blocks, use if or else then use { put your code inside here }
For example we can use the boolean from the previous example, this code is saying that if coinTails is true execute the code in the if brakcets otherwise execute code in the else brackets.

if (coinTails){
System.out.println("The coin flipped tails");
} else {
System.out.println("The coin flipped heads");
}

//Remember you need a boolean statement to trigger the if statement.

We will talk about the "for" and "while" loop

Arrays

Arrays are like storage containters, if we use variables as boxes, we can think of arrays as boxes we can store boxes in. //We need to use square brackets for arrays, e.g. int myNum[];
int myNum[] = {1,2,3,4,5};
//myNum contains the elements 1,2,3,4,5. To access these elements we need to put the index in the square bracket, starting at 0
System.out.println(myNum[0]);
